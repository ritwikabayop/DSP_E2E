⚡ How to apply any theme — 2 steps

Because all hardcoded colours were replaced with CSS variables in the last session, you only need to edit the :root block in src/index.css.

/* Example — switching to Teal Storm */
:root {
  --bg-base:       #060f0f;
  --bg-surface:    #081414;
  --bg-card:       #0c1e1e;
  --bg-sidebar:    #040c0c;
  --bg-header:     #081414;
  --border:        #0d2828;
  --border-subtle: #0d2828;
  --accent:        #2dd4bf;
  --accent-dim:    #0f9488;
  --accent-glow:   rgba(45,212,191,0.15);
}

Step 2: Optionally update the Ant Design dark token in src/App.jsx — the colorPrimary token inside DARK_THEME.
